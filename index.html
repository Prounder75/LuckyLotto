On this web page you will find a program I wrote that is designed to allow a user to input numbers, is this case lottery numbers, which would be compared against past lottery drawings and produce results that show if any or all the numbers match. The results will list the number of times one number would have hit; it will display the number of times two numbers would have hit, three numbers etc. up to the jaw dropping six number combination that could have won over $100 million for games like PowerBall and MegaMillions.

This was program was written a few years back and the database (is this case arrays) hasn't been updated to reflect all recent drawings so the progarm should be used for actual research on lottery number. I am currently using this program a way to demostrate one example of where I develope an algorithm for data analysis.
